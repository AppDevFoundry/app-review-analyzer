App Review Analyzer helps turn app store reviews into clear product insights.

Built on the [Next SaaS Stripe Starter](https://github.com/mickasmt/next-saas-stripe-starter), it integrates Next.js 14, Prisma, Neon, Auth.js v5, Resend, React Email, Shadcn/ui, and Stripe.

This documentation is your go-to resource for configuring and using the application effectively.

Let's get started and happy coding!
